# **App Name**: Legacy Locker

## Core Features:

- Camouflaged Interface: Appears as a fully functional calculator to outside observers.
- Secure Digital Vault: Allows users to store videos, audios, texts, and photos invisibly and securely.
- Recipient Management: Enables users to manage and assign digital memories to multiple recipients.
- Automated Check-in System: Includes manual check-in options and automatic inactivity detection to trigger memory delivery.
- Double Confirmation: Ensures a second layer of verification before delivering memories to prevent accidental releases, and uses the LLM as a tool to confirm delivery details with the user.
- Subscription and Payments: Manages user subscriptions through Mercado Pago, Google Play Billing, and PayPal, with automated payment processing and account suspension.
- Web Page for Recipients: Provides a secure, responsive web interface for recipients to view and download delivered memories.
- Firestore integration: Integrates Firestore for user authentication, database storage, and managing user profiles, memories, recipients, check-ins, and payments.

## Style Guidelines:

- Primary color: Deep, muted blue (#3B5998) to evoke a sense of security, seriousness, and trustworthiness. Chosen to align with the app's purpose of protecting digital legacies.
- Background color: Light, desaturated blue (#E8F0FE), providing a gentle backdrop that does not distract from the app's content.
- Accent color: Warm orange (#FFB347), offering a contrasting highlight for important calls to action.
- Body font: 'Inter', a grotesque-style sans-serif, provides a modern, neutral look suitable for both headlines and body text. Its clean appearance ensures readability and consistency throughout the app.
- Simple, clean icons that are intuitive and easy to understand, contributing to a user-friendly interface. Use icons related to security, memory, and legacy.
- Clean and organized layout, with clear separation of sections to help users navigate the app easily.
- Subtle transitions and animations to enhance the user experience without being distracting.
